La directory Common contiene classi che semplificano lo sviluppo di 
applicazioni. 

Le classi nella directory Common fanno parte del progetto e possono essere 
migliorate ulteriormente per soddisfare esigenze specifiche. È necessario 
prestare attenzione quando si modificano i metodi e le proprietà esistenti, in 
quanto le modifiche incompatibili richiedono di aggiornare di conseguenza il 
codice incluso in una serie di modelli di Visual Studio. Ad esempio, le pagine 
ulteriori aggiunte al progetto vengono scritte supponendo che i metodi e le 
proprietà originali nelle classi Common siano ancora presenti e che i nomi dei 
tipi non siano stati modificati.
